# Personal Portfolio — Muhammad Hammad

A responsive, modern personal portfolio static site. Drop these files into any web server or open `index.html` directly in a browser.

What's included:
- `index.html` — main page with About, Skills, Education, Experience, Contact sections.
- `styles.css` — responsive modern styles.
- `script.js` — mobile navigation + smooth scrolling.
- `avatar.svg` — placeholder avatar used in the hero card.

Replace the avatar:
- To use your own professional photo, save the image as `profile.jpg` and place it in the project root (`/Users/apple/Desktop/Frontend Internship/profile.jpg`). The site will automatically load that image in the hero card.

How to run:

Open the project folder and double-click `index.html` or run a quick local server, for example (macOS / zsh):

```bash
# using Python 3 built-in server
python3 -m http.server 8000
# then open http://localhost:8000 in your browser
```
